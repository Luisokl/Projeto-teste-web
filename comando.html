JavaScript

O que é JavaScript ?

De modo geral , o JavaScript é uma linguagem que suporta a progamação orientada a objetos. Ainda que não possua várias características desta orientação nativamente, é possivel realizar algumas emulações.

O JavaScript surgiu nos com o crescimento da internet no início dos anos 90. A empresa Netscape sentiu a necessidade de melhorar a interação do usuário com as páginas de navegador, que levava o mesmo nome. Então criou o LiveScript uma linguagem simples, que permitia a execução de escripts contidos no própio software.

Quais são as aplicações ?

  • Adcionar comportamento interativo às páginas da web,
  • Criar aplicativos web e móveis,
  • Criar servidores web e desenvolver aplicativos de servidor,
  • Desenvolver jogos.
